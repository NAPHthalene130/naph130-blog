## 缘起

一直想拥有一个完全由自己掌控的博客。不依赖任何后端服务，不依赖任何第三方平台，所有的内容都以 Markdown 文件的形式存在，可以随时迁移、备份、版本控制。

## 技术选型

这个博客使用了以下技术栈：

- **Vue 3**：渐进式 JavaScript 框架
- **Vite**：下一代前端构建工具
- **TypeScript**：类型安全的 JavaScript 超集
- **Tailwind CSS**：实用优先的 CSS 框架
- **markdown-it**：灵活的 Markdown 解析器

## 设计理念

### 纯客户端渲染

所有文章数据在浏览器端通过 HTTP 请求获取，无需任何服务端渲染或构建时生成 HTML。这使得部署变得极其简单——只需要一个静态文件服务器。

### 毛玻璃设计

采用了现代毛玻璃风格（Glassmorphism），灵感来源于 macOS 和 iOS 的设计语言。半透明的卡片、柔和的阴影、精致的圆角，营造出轻盈、通透的视觉体验。

### 多语言支持

内置中英文双语支持，通过 URL 路径前缀区分语言，UI 文本和文章内容完全分离，方便扩展更多语言。

## 功能一览

- 浅色/暗色模式切换
- 文章目录导航 (TOC)
- 客户端模糊搜索
- 代码块语法高亮与一键复制
- 图片灯箱预览
- 阅读进度条
- 页面过渡动画

## 未来计划
