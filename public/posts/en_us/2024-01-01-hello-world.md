## Origin

I have always wanted a blog that is completely under my control. No backend dependencies, no third-party platforms. All content exists as Markdown files — easily portable, backup-friendly, and version-controlled.

## Tech Stack

This blog is built with:

- **Vue 3**: The Progressive JavaScript Framework
- **Vite**: Next Generation Frontend Tooling
- **TypeScript**: JavaScript with syntax for types
- **Tailwind CSS**: A utility-first CSS framework
- **markdown-it**: Flexible Markdown parser

## Design Philosophy

### Pure Client-side Rendering

All article data is fetched via HTTP requests in the browser. No server-side rendering or build-time HTML generation required. This makes deployment extremely simple — just a static file server.

### Glassmorphism Design

Inspired by macOS and iOS design language — semi-transparent cards, soft shadows, and refined rounded corners create a light and airy visual experience.

### Multi-language Support

Built-in Chinese and English bilingual support with URL path prefix language detection. UI text and article content are completely separated for easy extension.

## Features

- Light/Dark mode toggle
- Table of Contents navigation
- Client-side fuzzy search
- Code syntax highlighting with one-click copy
- Image lightbox preview
- Reading progress bar
- Page transition animations

## Future Plans

Many ideas are still waiting to be implemented — RSS feeds, comment system, more themes... one step at a time.
